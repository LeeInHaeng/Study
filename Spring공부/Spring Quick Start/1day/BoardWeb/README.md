# 초기 프로젝트 생성
- Eclipse Marketplace에서 STS 설치
  - 해당 프로젝트에서는 Spring Tools 3 Add-on 설치
- Spring Legacy Project로 프로젝트 생성
  - Templates는 Spring MVC Project로 선택
- 프로젝트의 Properties에서 Project Facets 선택
  - Java 버전을 설치했던 버전으로 변경
  - Runtimes 탭에서 Apache Tomcat 버전 체크 후 Apply
- 프로젝트 파일 정리
  - src/main/resources 폴더에 log4j.xml을 제외한 나머지 파일 모두 삭제
  - src/main/webapp/WEB-INF 아래에 spring과 views 폴더 삭제
  - src/main/webapp/WEB-INF/web.xml에서 루트 엘리먼트를 제외한 나머지 설정 모두 삭제
  - pom.xml 파일에서 properties 속성의 springframework-version을 최신 버전으로 변경
