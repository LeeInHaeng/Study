# 후기
알고리즘의 경우에는 컴퓨터공학과이다보니 많이 어렵지 않았던 것 같습니다.
업무프로세스는 어떤 실제 사례를 도형으로 정리하는 것이 아니라 개념을 물어보는 것이다 보니 만약 2015년 기출문제부터 최신 기출문제까지 정리를 했더라면 맞출 수 있지 않았을까 라는 아쉬움이 남았습니다. 
이런식으로 개념을 물어보는 업무프로세스의 경우에는 영어를 잘하시는 분이라면 충분히 유추해서 맞출 수 있을 것 같습니다. 실제로 시험 문제의 경우에는 영어 약자로 몇 글자이고, 한글로 해석하면 이런 뜻이다. 라는 것도 알려주기 때문에 어느정도 유추해볼 수 있습니다.
전산영어의 경우에는 따로 공부를 하지 않았지만, 많이 어렵지 않은 개념을 영어로 내는 것이기 때문에 해석을 하시면 충분히 풀 수 있을것 같습니다. 이번에 DRM의 경우에도 최근 기출문제 중 한글로 적혀있는 개념을 영어로 바꾼 것이었습니다.
짧은 기간 공부하다보니 2017년과 2018년 기출문제밖에 정리하지 못하고 시험을 봤는데, 조금 더 기간을 길게 잡으실 분들은 더 많은 기출문제들을 정리하시면 훨씬 좋은 점수를 받을 수 있을것 같습니다.
