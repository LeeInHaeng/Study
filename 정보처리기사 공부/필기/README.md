공부 방법은 한글 문서에 정리 하는 식으로 공부를 했다.
주로 기출문제의 보기들을 기반으로 정리 했는데 예를 들면 옳지 않은 답을 찾는 문제가 있으면 옳지 않은 답 1개를 제외한 나머지 3개의 옳은 답을 한글 문서에 정리하는 식으로 공부를 진행했다. 2013년 8월 기출문제부터 2018년 4월 기출문제까지 정리를 진행했고, 해설지가 나와있지 않은 경우에는 시나공 책을 참고 하거나, 인터넷 검색을 통해 부족한 부분들을 채워 넣었으며 해설지가 있는 기출문제의 경우에는 해설지를 바탕으로 정리를 진행했다. 기출 문제의 경우에는 http://www.comcbt.com/xe/j4 해당 사이트에서 다운로드 했다.
아무래도 보기들을 옮겨 적는식의 정리를 하다 보니 양이 많이 나왔다. (한글 문서로 44쪽)
정리한 양이 너무 많다 보니 다 외우고 시험을 보러 가지는 못했지만, 정리 하면서 어느정도 암기가 되었다.
